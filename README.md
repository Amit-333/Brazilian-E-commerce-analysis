# Brazilian-E-commerce-analysis

## Project Overview
Brazilian e-commerce sales analysis with SQL, Python, data visualization, and business insights.

## Tools Used
- SQL Server
- Python
- Pandas
- Matplotlib
- Google Colab

## Business Questions

- Which states generate the most revenue?
- Which categories drive most sales?
- How strong is customer retention?
- How concentrated is revenue?
- How reliable are deliveries?

## Key Findings

- Top 10 categories contribute 62.36% of revenue.
- Top 25 categories contribute ~90% of revenue.
- Customer retention is only 2.16%.
- Repeat customers generate 1.88x more revenue.
- Over 99% of deliveries arrive on time.

## Dataset

This project uses four tables from the Brazilian E-Commerce Public Dataset by Olist:

- Customers
- Orders
- Order Items
- Products

These tables were used to analyze customer behavior, revenue trends, product performance, and operational efficiency.
