# Brazilian-E-commerce-analysis

## Project Overview
Brazilian e-commerce sales analysis with SQL, Python, data visualization, and business insights.

## Tools Used
- SQL Server
- Python
- Pandas
- Matplotlib
- Google Colab

## Business Questions

- Which states generate the most revenue?
- Which categories drive most sales?
- How strong is customer retention?
- How concentrated is revenue?
- How reliable are deliveries?

## Key Findings

- Top 10 categories contribute 62.36% of revenue.
- Top 25 categories contribute ~90% of revenue.
- Customer retention is only 2.16%.
- Repeat customers generate 1.88x more revenue.
- Over 99% of deliveries arrive on time.

## Dataset

This analysis was conducted using four files from the Brazilian E-Commerce Public Dataset by Olist:

|     Table   | Description |
|-------------|-------------|
| Customers   | Customer information and location data |
| Orders      | Order status and purchase timestamps |
| Order Items | Product-level sales and revenue information |
| Products    | Product category and product attributes |

The four tables were joined using customer IDs, order IDs, and product IDs to perform end-to-end business analysis.
